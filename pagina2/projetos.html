Projeto FlutterFlow

Este projeto foi feito durante as férias do SENAI, é um e-commerce de Softwares e OS. Criei ele usando FlutterFlow que é uma ferramenta Low Code. Este projeto consta com dois bancos de dados, a parte dos produtos foi feita usando o Supabase, já na parte de autenticação foi usado o Firebase. Utilizei a API do Github e Google e implementei para serem opções do login do aplicativo, utilizei API Calls, Data Types, Firestore, App Values, Gemini Chat e Purchases que o aplicativo oferece.
